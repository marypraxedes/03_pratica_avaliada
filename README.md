# Prática Avaliada 03 - Programação Orientada a Objetos (Java)

## Descrição

Este repositório contém as atividades práticas desenvolvidas em Java utilizando os principais conceitos de **Programação Orientada a Objetos (POO)**.

Durante as atividades foram aplicados conceitos como:

- Classes e objetos
- Encapsulamento
- Construtores
- Herança
- Polimorfismo
- Sobrescrita de métodos
- Tratamento de exceções
- Validação de dados

## Atividades Desenvolvidas

O projeto é composto por três sistemas:

### Sistema de Usuários

Sistema para gerenciamento de usuários utilizando herança e polimorfismo.

Funcionalidades:
- Cadastro de usuários com nome e email.
- Criação de administradores com nível de acesso.
- Validação de dados informados.
- Tratamento de erros através de exceções.

Principais classes:
- `Usuario`
- `Administrador`
- `Validacao`
- `TestaUsuarios`

---

### Sistema de Veículos

Sistema para gerenciamento de veículos elétricos.

Funcionalidades:
- Cadastro de carros com modelo e velocidade.
- Controle de bateria de carros elétricos.
- Cálculo de consumo de bateria durante acelerações.
- Validação de aceleração e nível de bateria.
- Tratamento de exceções.

Principais classes:
- `Carro`
- `CarroEletrico`
- `Validacao`
- `TestaCarros`

---

### Sistema de Notificações

Sistema para envio de notificações utilizando diferentes tipos de comunicação.

Funcionalidades:
- Envio de notificações por email.
- Envio de notificações por SMS.
- Validação de mensagens, emails e telefones.
- Uso de herança e polimorfismo para diferentes tipos de notificações.
- Tratamento de erros.

Principais classes:
- `Notificacao`
- `NotificacaoEmail`
- `NotificacaoSMS`
- `Validacao`
- `TestaNotificacoes`

---

## Organização do Projeto

O projeto está dividido em três packages, cada um contendo um sistema desenvolvido com conceitos de Programação Orientada a Objetos:

- **atividade01**: Sistema de Usuários  
  Implementação de usuários e administradores utilizando herança, encapsulamento, validações e tratamento de exceções.

- **atividade02**: Sistema de Veículos  
  Implementação de carros e carros elétricos utilizando herança, polimorfismo, controle de bateria e validações.

- **atividade03**: Sistema de Notificações  
  Implementação de notificações por email e SMS utilizando herança, sobrescrita de métodos, validação de dados e tratamento de exceções.
